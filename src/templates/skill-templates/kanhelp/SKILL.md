# Kanhelp Skill

Shows comprehensive help for the Claude Kanban system, including:
- Available skills and commands
- Native tools integration
- Status mapping
- Directory structure
- Card format
- Session recovery
- Tips and resources

When invoked, this skill executes:
`node .kanhelper/dist/skills/kanhelp.js`

## Usage

Simply type `/kanhelp` in Claude Code to view the complete Kanban help guide.

## What It Shows

- **Skills**: /kanboard, /kansync, /kanhelp
- **Native Tools**: TaskCreate, TaskUpdate integration
- **Status Mapping**: How task statuses map to Kanban folders
- **Directory Structure**: Visual representation of folder layout
- **Card Format**: Example of markdown card with frontmatter
- **Session Recovery**: How to resume work across sessions
- **CLI Commands**: Installation and verification commands
- **Tips**: Best practices and helpful hints
- **Resources**: Links to documentation and config files
